# 更新日志

本文档记录公开发布版本的主要变化。日期采用北京时间。

## v3.2.3 - 2026-05-24

- 更新应用版本号到 `v3.2.3`。
- 优化工具列表与收藏入口的流畅体验，完善虚拟列表、异步路径状态和图标解析缓存。
- 增强运行时数据体检，明确区分模板数据与 `.runtime/data` 实际用户数据。
- 拆分路径状态、图标解析、导入边界和主窗口控制器，降低后续迭代回归风险。
- 加固一键更新、发布包校验和打包流程，发布包继续保留用户 `.runtime` 数据边界。
- 同步更新 README、发布检查清单和预置工具拆分镜像。

## v3.2.2 - 2026-05-02

- 更新应用版本号到 `v3.2.2`。
- 优化新建工具与编辑工具窗口的视觉样式、尺寸和内容布局。
- 在工具卡片底部增加收藏与打开笔记快捷按钮，减少右键菜单依赖。
- 新增批量删除工具和删除所有工具入口，方便整理工具库。
- 优化工具卡片响应式布局，提升固定窗口下的空间利用率。
- 加固运行时数据、配置备份、日志查看和发布包构建流程。
- 完善 README、用户指南、发布检查清单和发布包版本说明。
